# Projeto Vagas SouJúnior
Projeto de ETL de vagas para júnior em TI realizado pela SouJúnior

### Criar Ambiente
#### Ubuntu
`python3 -m venv venv`
#### Windows
`python -m venv venv`

### Instalar Dependências
`pip install -r requirements.txt`

### Setup Chromedriver 
#### Ubuntu
`sudo apt-get install chromium-chromedriver` 
#### Windows

### ETL
`cd etl-vagas`
`python execute.py`
